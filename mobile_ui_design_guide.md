#  UI 设计规范

## 1. 设计基准

设计尺寸以移动端为主，推荐基准画布为：

- iPhone：375 × 812
- 安卓通用：360 × 800
- 小程序页面宽度按 375px 设计
- 所有内容需适配 320px–430px 宽度屏幕

页面整体风格要求：

- 简洁、清晰、年轻化
- 层级明确
- 操作区域易点击
- 内容留白充分
- 避免文字拥挤
- 避免过度装饰

---

## 2. 字体规范

默认字体：

- iOS：PingFang SC
- Android：HarmonyOS Sans / MiSans / Noto Sans SC
- Web fallback：Microsoft YaHei, Arial, sans-serif

### 字号层级

| 类型 | 字号 | 字重 | 使用场景 |
|---|---:|---:|---|
| 页面主标题 H1 | 22px | 600 / 700 | 页面标题、核心模块标题 |
| 一级标题 H2 | 20px | 600 | 首页模块标题、详情页大标题 |
| 二级标题 H3 | 18px | 600 | 卡片标题、分组标题 |
| 三级标题 H4 | 16px | 500 / 600 | 列表标题、表单标题 |
| 正文 Text | 14px | 400 | 普通说明、卡片内容 |
| 辅助文字 | 13px | 400 | 摘要、副标题、状态说明 |
| 弱提示文字 | 12px | 400 | 标签、时间、备注 |
| 极小文字 | 10px–11px | 400 | 角标、数字徽标、辅助标识 |

## 4. 间距与边界规范

采用 4px 栅格体系。

### 页面边距

| 类型 | 数值 |
|---|---:|
| 页面左右安全边距 | 16px |
| 紧凑页面边距 | 12px |
| 大卡片外边距 | 16px |
| 卡片内边距 | 12px–16px |
| 模块上下间距 | 20px–24px |
| 标题与内容间距 | 8px–12px |
| 列表项间距 | 8px–12px |

---

## 5. 圆角规范

| 组件 | 圆角 |
|---|---:|
| 页面大卡片 | 16px |
| 普通卡片 | 12px |
| 小卡片 | 10px |
| 按钮 | 12px |
| 输入框 | 10px–12px |
| 标签 Tag | 6px–8px |
| 图片封面 | 10px–12px |
| 弹窗 / 抽屉 | 20px |

---

## 6. 按钮规范

### 主按钮

- 高度：44px–48px
- 圆角：12px
- 背景色：#2563EB
- 文字颜色：#FFFFFF
- 字号：16px
- 字重：600

### 次按钮

- 高度：40px–44px
- 背景色：#EFF6FF
- 文字颜色：#2563EB
- 圆角：12px

---

## 7. 输入框规范

输入框高度：

- 普通输入框：44px
- 搜索框：40px–44px
- 多行文本框：96px–140px

样式：

- 背景色：#F9FAFB
- 边框：1px solid #E5E7EB
- 圆角：10px–12px
- 内边距：12px–14px
- 字号：14px

---

## 8. 卡片规范

### 普通内容卡片

- 背景：#FFFFFF
- 圆角：12px–16px
- 内边距：12px–16px
- 卡片间距：12px

### 瀑布流卡片

- 两列布局
- 列间距：10px–12px
- 卡片上下间距：12px
- 卡片宽度自适应
- 高度可以不同，但卡片边界必须端正

---

## 9. 标签 Tag 规范

标签高度：

- 小标签：22px
- 普通标签：24px–26px

样式：

- 圆角：6px–8px
- 左右内边距：8px
- 字号：12px

---

## 10. 顶部导航规范

高度：

- 普通导航栏：44px
- 含状态栏整体高度：88px 左右

内容规则：

- 左侧：返回 / 标题 / Logo
- 中间：页面标题
- 右侧：操作按钮、消息、更多

---

## 11. Banner 规范

Banner 高度：

- 小 Banner：120px
- 标准 Banner：140px–160px
- 大视觉 Banner：180px–220px

样式建议：

- 圆角：16px–20px
- 内边距：16px–20px
- 主标题建议 20px–24px
- 副标题建议 13px–14px

---

## 12. 弹窗与底部抽屉

### 弹窗

- 宽度：屏幕宽度的 80%–88%
- 圆角：20px
- 内边距：20px

### 底部抽屉

- 顶部圆角：20px–24px
- 内边距：16px
- 最大高度：屏幕高度的 80%

---

## 13. 页面层级规范

页面建议分为 4 层：

1. 页面背景层
2. 内容模块层
3. 卡片 / 表单层
4. 浮层 / 弹窗 / Toast 层

---

## 14. 出血与适配规范

- 页面背景色必须铺满全屏
- 图片 Banner 可满宽，但内容需保留 16px 安全边距
- 底部固定栏需适配安全区
- 不允许核心文字贴边
- 不允许横向滚动

---


---

## 16. Codex 开发约束

1. 页面左右默认保留 16px 安全边距。
2. 所有可点击区域不得小于 44px × 44px。
3. 字号不得小于 12px。
4. 卡片圆角统一使用 12px 或 16px。
5. 页面背景统一使用 #F5F7FA。
6. 主要按钮统一使用 #2563EB。
7. 标签最多展示 3 个。
8. 移动端优先，PC 端只做兼容。
