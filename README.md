# 海龟汤评价管理系统 Demo

一个轻量化、手机端优先的海龟汤发布与评价系统。

## 快速启动

```bash
cp .env.example .env
docker compose up -d mysql
npm install
npm run dev
```

- 前端：http://localhost:5173
- 后端：http://localhost:4000
- 默认管理员：admin / asd110035

## 功能

- 注册、登录、退出，登录状态持久化 30 天。
- 发布海龟汤，支持设置是否公开汤面、是否公开汤底。
- 列表筛选：标题、作者、类型、均分区间、是否公开汤底。
- 详情页查看汤面、汤底、主持人手册。
- 一人一评，总分手动，六维评分可选。
- 汤底/主持人手册申请查看，作者和管理员审批。
- 站内消息提醒。
- 详情内容导出 PNG。
